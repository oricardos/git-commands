Comandos Git

Comando Descrição
git branch <nome-da-branch> Cria uma nova branch com o nome especificado.
git checkout <nome-da-branch> Altera a branch atual para a branch especificada.
git status Mostra o status atual do repositório, indicando arquivos modificados e não commitados, arquivos adicionados, arquivos removidos, etc.
git commit -m "Mensagem do commit" Salva as mudanças adicionadas ao "stage" com uma mensagem descritiva do commit.
git push <nome-remoto> <nome-da-branch> Envia as mudanças na branch especificada para um repositório remoto com o nome especificado.
git pull <nome-remoto> <nome-da-branch> Atualiza a branch atual com as mudanças da branch remota com o nome especificado.
